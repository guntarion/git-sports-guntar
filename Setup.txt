Kamu perlu memilih salah satu. Panduan credentials masing-masing:

---
PATH A: Setup Strava

A1. Buat Strava API App

1. Buka https://www.strava.com/settings/api
2. Buat aplikasi baru, isi:
 - Application Name: bebas (misal git-sweaty)
 - Category: Other
 - Authorization Callback Domain: localhost
3. Catat Client ID dan Client Secret

A2. Dapatkan Refresh Token

Buka URL ini di browser (ganti YOUR_CLIENT_ID):
https://www.strava.com/oauth/authorize?client_id=YOUR_CLIENT_ID&response_type=code&redirect_uri=http%3A%2F%2Flocalhost%2Fexchange_token&ap
proval_prompt=force&scope=read%2Cactivity%3Aread_all
- Klik Authorize
- Kamu akan redirect ke localhost yang error (normal) — salin nilai code dari URL

A3. Tukar Code → Refresh Token

curl -sS -X POST https://www.strava.com/oauth/token \
 -d client_id=YOUR_CLIENT_ID \
 -d client_secret=YOUR_CLIENT_SECRET \
 -d code=YOUR_CODE \
 -d grant_type=authorization_code
Dari response JSON, salin nilai refresh_token.

A4. Set GitHub Secrets

gh secret set STRAVA_CLIENT_ID --body "YOUR_CLIENT_ID" --repo guntarion/git-sports-guntar
gh secret set STRAVA_CLIENT_SECRET --body "YOUR_CLIENT_SECRET" --repo guntarion/git-sports-guntar
gh secret set STRAVA_REFRESH_TOKEN --body "YOUR_REFRESH_TOKEN" --repo guntarion/git-sports-guntar

  Kemungkinan penyebab:
  - Endpoint China (dashscope.aliyuncs.com) dipakai tapi key-mu dari Alibaba Cloud International (dashscope-intl). Sudah saya revert ke
  -intl.
  - Atau key-nya sendiri salah. Cek kembali QWEN_API_KEY di Settings → Secrets → Actions.

  Untuk mendapatkan API key yang valid:
  1. Buka https://dashscope.console.aliyun.com/ (atau -intl untuk international)
  2. Buka API Keys → Create/copy key
  3. Set di repo: Settings → Secrets → Actions → QWEN_API_KEY

  Cara trigger manual Sync

  Via GitHub UI:
  1. Buka repo → tab Actions
  2. Klik workflow "Sync Heatmaps" di sidebar kiri
  3. Klik "Run workflow" → pilih source (garmin), full_backfill (true/false) → Run

  Via CLI (dari terminal ini):
  # Sync biasa
  gh workflow run "Sync Heatmaps" --field source=garmin

  # Full backfill (re-fetch semua data)
  gh workflow run "Sync Heatmaps" --field source=garmin --field full_backfill=true
